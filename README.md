Udemy Clone

A simple Udemy Clone website built using HTML and CSS. This project is a front-end clone of the Udemy homepage, created to practice web design and responsive layouts.

Features
Responsive homepage
Navigation bar
Course cards
Categories section
Footer
Clean and simple UI
Technologies Used
HTML5
CSS3

Project Structure
Udemy-Clone/
│── index.html
│── style.css
└── images/

How to Run
Download or clone this repository.
Open the project folder.
Open index.html in your browser.
